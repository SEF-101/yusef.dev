# Free Audiobooks Collection

I maintain a collection of free audiobooks that you can access through my server.

## How to Connect

### Web Browser
1. Visit [books.sef101.com](https://books.sef101.com) in your web browser
2. Browse the available titles by category or use the search function
3. Click on any audiobook to stream directly from your browser
4. You can also download titles for offline listening

### Mobile Access
For the best mobile experience, you can use the Plappa app:

1. Download the Plappa app from your device's app store
2. When prompted for a server URL, enter: `books.sef101.com`
3. Log in with your credentials
4. Browse and listen to audiobooks directly from your mobile device

## Account Access

If you need a login account or are having trouble accessing the server, please contact me directly.

## Request Audiobooks

Don't see an audiobook you're looking for? Let me know! I'm happy to add new titles to the collection when possible.
